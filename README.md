# A dynamic Tunnel Digital Twin Framework for Real-Time Situation Awareness and Emergency Response

This repository contains the code and resources for our paper `"Vision-Driven Dynamic Tunnel Digital Twin for Real-Time Situation and Emergency Awareness"` published in `[Underground Space]`. The framework integrates real-time vision data from CCTV cameras to create a real-time virtual representation of tunnel environments, enabling enhanced situation awareness and effective emergency response.
